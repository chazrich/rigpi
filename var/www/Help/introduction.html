RigPi™ Station Server v 2.0

 

 

RigPi Station Server (RSS) is a computer system designed to control your station and facilitate on-the-air activities. RSS introduces a radically new way of station control by using a server model for allowing multiple users and radios to be accommodated simultaneously through standard Internet browsers.
